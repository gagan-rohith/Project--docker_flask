Pytest

The pytest framework makes it easy to write small tests, yet scales to support complex functional testing for applications and libraries.

Simple pages involving bootstrapping were created. Carousel style of images was used.Testing was done using pytest.

Outline of Directory

docker-compose.yml is a file that contains the configuration to develop locally.

Dockerfile is used to run the flask development server instead of running the server that is used for hosting on Heroku.

pytest.ini is a pytest file used to initialize program settings.
